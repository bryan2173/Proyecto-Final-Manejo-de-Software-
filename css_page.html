Curso CSS Cascading Style Sheets

CSS es el lenguaje utilizado para dar estilo y formato a las páginas web creadas con HTML. Con CSS, puedes aprender a personalizar el diseño visual de un sitio, definiendo colores, tipografías, márgenes, espaciados, alineaciones y tamaños. Además, te permite crear diseños responsivos adaptados a diferentes dispositivos moviles

Nivel Básico: Fundamentos de CSS

Objetivo: Aprender a aplicar estilos básicos y familiarizarse con los conceptos iniciales.

1- Introducción a CSS

  • ¿Qué es CSS? Una herramienta para separar la estructura (HTML) del diseño.
  • Métodos para incluir CSS.
  • Inline: Dentro de una etiqueta HTML usando el atributo style..
  • Interno: Dentro de etiquetas "style" en el "head"
  • Externo: En un archivo .css enlazado con "link".

2- Selectores Básicos

  • Selectores de elementos: Aplican a todas las etiquetas de un tipo (ej. p {}).
  • Selectores de clase: Utilizan . seguido del nombre (ej. .boton {}).
  • Selectores de ID: Utilizan # seguido del nombre (ej. #encabezado {}).

3- Colores y Fondos

  • Definir colores en CSS con:
  • Palabras clave (red, blue).
  • Valores hexadecimales (#FFFFFF).
  • Valores RGB y HSL
  • Estilización de fondos
  • background-color.
  • Imágenes de fondo con background-image.

4- Propiedades de Texto

  • Estilización: font-family, font-size, font-weight, color
  • Alineación: text-align.
  • Control de espaciado: line-height, letter-spacing.

5- Modelo de Caja (Box Model)

  • Estructura de los elementos:
  • Contenido.
  • Relleno (padding).
  • Bordes (border).
  • Márgenes (margin).

Nivel Intermedio: Diseño y Organización

Objetivo: Diseñar páginas más estructuradas y funcionales.

1. Posicionamiento

  • Propiedad position:static, relative, absolute, fixed, sticky.
  • Uso del z-index para capas.

2- Diseño Responsivo

  • Uso de media queries para adaptar estilos según el tamaño de pantalla.
  • Meta etiqueta viewport para optimizar el diseño móvil.

3- Pseudo-clases y Pseudo-elementos

  • Pseudo-clases: :hover, :focus, :nth-child.
  • Pseudo-elementos: ::before, ::after.

Nivel Avanzado: Técnicas Modernas y Herramientas

Objetivo: Implementar diseños profesionales y optimizados con técnicas avanzadas.

1. Flexbox

  • Diseño flexible para distribución de elementos.
  • Propiedades clave
  • justify-content
  • align-items.
  • flex-wrap.

2- Grid Layout

  • Creación de estructuras con grid-template-rows y grid-template-columns.
  • Uso de áreas definidas con grid-area.

3- Animaciones y Transiciones

  • Crear transiciones suaves entre estados con transition.
  • Animaciones avanzadas con @keyframes.

4- Variables CSS y Preprocesadores

  • Uso de variables CSS (--variable) y funciones como var().
  • Introducción a preprocesadores: SASS y LESS.
