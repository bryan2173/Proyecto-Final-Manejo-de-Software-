Curso HTML (HyperText Markup Language)

HTML es el lenguaje estándar para estructurar y presentar contenido en la web. Con HTML, puedes aprender a crear la estructura básica de páginas web, incluyendo encabezados, párrafos, listas, enlaces, imágenes, tablas y formularios,incorporar elementos multimedia como audio y video, organizar

Nivel Básico: Introducción y Fundamentos

Objetivo: Familiarizarse con el lenguaje HTML, comprender su estructura y construir páginas sencillas.

1- Introducción a HTML

  • ¿Qué es HTML y para qué se usa?
  • Instalación de editores de texto (Visual Studio Code, Sublime Text, etc.).
  • Estructura básica de una página web: html, head, body.

2- Etiquetas básicas de contenido

  • Texto: h1 a h6, p, br. /b /li
  • Listas: ul, ol, li.
  • Enlaces: a y rutas relativas/absolutas.
  • Imágenes: img y atributos básicos (src, alt).

3- Formato básico del texto

  • Uso de strong, em, b, i if, else, elif.
  • Entidades HTML: caracteres especiales como &_nbsp y &_copy.

Nivel Intermedio: Organización y Componentes

Objetivo: Aprender a estructurar mejor las páginas, manejar tablas, formularios, y multimedia.

1. Tablas en HTML

  • Estructura básica: table>, tr>, td>, th.
  • Atributos: colspan, rowspan, border.
  • Estilización básica con CSS.

2- Formularios y entradas de datos

  • Etiquetas principales: form, input, textarea, select
  • Tipos de entrada: texto, contraseñas, botones, casillas de verificación.
  • Envío de datos y manejo básico con atributos como action y method.

3- Inclusión de multimedia

  • Uso de audio y video para contenido multimedia.
  • Propiedades como controls, autoplay, y formatos compatibles.
  • Incorporación de iframe para contenido embebido (Google Maps, YouTube).

4- Semántica y accesibilidad

  • Uso de etiquetas modernas: header, footer, main, section
  • Mejores prácticas para accesibilidad (atributos alt, estructura clara).

Nivel Avanzado: HTML5 y Desarrollo Profesional

Objetivo: AConstruir páginas web dinámicas y preparadas para la web moderna

1. HTML5 Semántico

  • Etiquetas avanzadas: article, aside, nav, figure/figure
  • Marcado semántico para mejorar SEO y accesibilidad.

2- APIs de HTML5

  • Geolocalización con navigator.geolocation.
  • Canvas para gráficos dinámicos con canvas..
  • Almacenamiento web: localStorage y sessionStorage.

3- Diseño responsivo con HTML y CSS

  • Configuración de la metaetiqueta meta viewport para móviles.
  • Uso de frameworks CSS como Bootstrap para diseño adaptable.

4- Proyectos prácticos

  • Página de portafolio profesional.
  • Formulario avanzado con validación dinámica.
  • Página interactiva con gráficos generados en canvas.
